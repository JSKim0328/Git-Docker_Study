크기(dimension)

CSS 스타일을 이용하여 HTML 요소의 크기를 조절할 수 있다.

가장 많이 쓰이는 크기의 단위는 다음과 같다.

  • 백분율 단위(%)
  • 배수 단위(em)
  • 픽셀 단위(px)
백분율 단위는 100%, 배수 단위는 1em을 기본 크기로 놓고, 그에 대한 상대적인 크기를 정할 수 있다.
픽셀 단위는 스크린의 픽셀을 기준으로 하는 절대적인 크기를 정할 수 있다.

요소의 크기를 지정할 수 있는 속성에는 다음과 같은 것들이 있다.

height & width

각각 요소의 높이와 너비를 지정할 수 있는 속성이다. 기본 설정값은 auto이다.

max-width

최대 너비를 지정하는 속성이다. 너비를 설정하지 않고 최대 너비만 설정했을 경우, 브라우저의 크기가 최대 너비보다 작아지면 자동으로 요소의 너비가 줄어든다는 특성이 있다. 너비가 설정되고 브라우저가 너비보다 작아질 경우의 행동은 overflow 속성을 통해 설정할 수 있다.

min-width

최소 너비를 지정하는 속성이다. 너비 속성을 지정하지 않으면 100%가 기본값이다. 웹 브라우저의 크기가 최소 너비보다 작아지면 스크롤바가 생성된다. 이 역시 overflow 속성으로 변경할 수 있다.

max-height

최대 높이를 지정하는 속성이다.
기본값은 none이며 요소의 높이가 최대 높이보다 커졌을 때의 동작은 overflow 속성으로 설정할 수 있다.

min-height

최소 높이를 지정하는 속성이다. 이는 max-height와 height 속성 보다 우선 적용된다.

박스 모델

모든 html 요소는 다음 그림과 같이 박스 모델로 구성된다.

1. 내용(content) : 텍스트나 이미지가 들어있는 박스의 실질적인 내용 부분입니다.
2. 패딩(padding) : 내용과 테두리 사이의 간격입니다. 패딩은 눈에 보이지 않습니다.
3. 테두리(border) : 내용와 패딩 주변을 감싸는 테두리입니다.
4. 마진(margin) : 테두리와 이웃하는 요소 사이의 간격입니다. 마진은 눈에 보이지 않습니다.

아웃라인(outline)

outline 속성은 border의 아웃라인 부분의 스타일을 설정한다.
다만 이 속성은 요소의 크기, 레이아웃에 영향을 주지 않는다.
